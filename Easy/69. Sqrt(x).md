Implement int sqrt(int x).

Compute and return the square root of x.

实现方法 int sqrt(int x)，给 x 开根号。

## 0x00
```java
public int mySqrt(int x) {
    long r = x;
    while (r*r > x)
        r = (r + x/r) / 2;
    return (int)r;
}
```
